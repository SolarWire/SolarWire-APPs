## 二、技术细节

### 2.1 样式统一化

**修改前**：
```css
/* 多处使用不同的透明度定义 */
background: color-mix(in srgb, var(--accent-color) 35%, transparent);
background: rgba(79, 195, 247, 0.05);
background: ${primaryColor}59; /* 内联样式 */
```

**修改后**：
```css
/* 统一使用预定义变量 */
background: var(--accent-opacity-35);
background: var(--accent-opacity-10);
```

### 2.2 内联样式消除

**PropertyPanel.tsx 修改前**：
```tsx
<button
  style={{
    padding: '4px 8px',
    fontSize: '11px',
    border: '1px solid var(--border-light)',
    borderRadius: '3px',
    background: mode === 'relative' ? 'var(--primary-color)' : 'var(--bg-light)',
    color: mode === 'relative' ? 'var(--white)' : 'var(--text-dark)',
    cursor: 'pointer'
  }}
>
```

**PropertyPanel.tsx 修改后**：
```tsx
<button className={mode === 'relative' ? 'active' : ''}>
```

**PropertyPanel.css 新增**：
```css
.coordinate-type-selector .toggle-group button.active {
  background: var(--accent-color);
  color: var(--bg-primary);
  border-color: var(--accent-color);
}
```

### 2.3 MonacoEditor 优化

**修改前**：
- 每次 `primaryColor` 变化都重新创建 style 标签
- 动态注入内联样式到 document.head

**修改后**：
- 使用 CSS 类 `.highlight-line`
- 通过 CSS 变量自动响应主题色变化
- 无需重新创建装饰器
