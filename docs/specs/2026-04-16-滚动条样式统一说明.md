# 滚动条样式统一说明

**目标**: 所有滚动条样式与 Monaco 编辑器完全一致

---

## Monaco 编辑器滚动条样式特征

1. **宽度/高度**: 10px
2. **圆角**: 无（直角）
3. **轨道背景**: `var(--bg-primary)` (#1e1e1e)
4. **滑块背景**: `var(--border-color)` (#3c3c3c)
5. **滑块悬停**: `var(--text-muted)` (#888888)
6. **转角背景**: `var(--bg-primary)` (#1e1e1e)

---

## 统一的 Scrollbar 组件样式

```css
/* 滚动条组件样式 - 与 Monaco 编辑器保持一致 */
.scrollbar {
  height: 100%;
  overflow: auto;
}

.scrollbar::-webkit-scrollbar {
  width: 10px;
  height: 10px;
}

.scrollbar::-webkit-scrollbar-track {
  background: var(--bg-primary);
}

.scrollbar::-webkit-scrollbar-thumb {
  background: var(--border-color);
  border-radius: 0; /* 直角，与 Monaco 编辑器一致 */
}

.scrollbar::-webkit-scrollbar-thumb:hover {
  background: var(--text-muted);
}

.scrollbar::-webkit-scrollbar-corner {
  background: var(--bg-primary);
}

/* Firefox 滚动条支持 */
.scrollbar {
  scrollbar-width: thin;
  scrollbar-color: var(--border-color) var(--bg-primary);
}
```

## 应用位置

所有需要滚动的视图和组件都应使用 `<Scrollbar>` 组件包裹：

### 视图组件
- ✅ FileView - 文件列表
- ✅ RequirementView - 需求文档列表
- ✅ SolarWireView - SolarWire 页面列表
- ✅ GitView - Git 状态和提交历史
- ✅ VersionView - 版本历史记录

### 编辑模式
- ✅ MarkdownPreview - Markdown 预览
- ✅ SolarWireMode - SolarWire 可视化编辑

### 其他组件
- ✅ PropertyPanel - 属性面板
- ✅ Note 编辑框 - 元素备注（高度自适应，由外层 Scrollbar 处理滚动）

## 关键 CSS 修改

### 移除所有子组件的 overflow: auto

**错误的做法** ❌:
```css
.parent {
  overflow: auto; /* 会产生默认滚动条 */
}

.child {
  overflow: auto; /* 多层嵌套 */
}
```

**正确的做法** ✅:
```css
.parent {
  overflow: hidden; /* 不产生滚动 */
}

.child {
  overflow: visible; /* 内容溢出可见 */
}

/* 只有 Scrollbar 组件有 overflow: auto */
.scrollbar {
  overflow: auto; /* 唯一正确的滚动容器 */
}
```
