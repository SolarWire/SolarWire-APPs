# 主色（Primary Color）应用规划

### 1.1 主色设置机制

**存储位置**：`src/app/stores/settingsStore.ts`

- 默认值：`#FCA506`（橙色）
- 存储方式：localStorage + CSS 变量 `--accent-color`
- 更新机制：修改时自动更新 `document.documentElement.style.setProperty('--accent-color', color)`

1. **坐标模式切换按钮** (`PropertyPanel.tsx`)
   - 当前：使用 `primaryColor` prop 但未在 CSS 中定义变量
   - 问题：硬编码在组件内联样式中
   - 建议：迁移到 CSS 变量，统一样式管理

   **修改方案**：
   ```css
   /* PropertyPanel.css */
   .coordinate-mode-button.active {
     background: var(--accent-color);
     color: var(--bg-primary);
   }
   ```

   ```tsx
   /* PropertyPanel.tsx - 移除内联样式 */
   <button 
     className={`coordinate-mode-button ${mode === 'relative' ? 'active' : ''}`}
     /* 移除 style={{ backgroundColor: mode === 'relative' ? primaryColor : '' }} */
   >
   ```

2. **Monaco 编辑器高亮行** (`MonacoEditor.tsx`)
   - 当前：内联样式注入 `${primaryColor}59`
   - 问题：每次颜色变化都重新创建装饰器，性能开销
   - 建议：使用 CSS 类 + CSS 变量

   **修改方案**：
   ```css
   /* MonacoEditor.css */
   .monaco-highlight-line {
     background-color: color-mix(in srgb, var(--accent-color) 35%, transparent) !important;
   }
   ```

   ```tsx
   /* MonacoEditor.tsx - 使用 CSS 类而非内联样式 */
   // 移除 decorations 中的 backgroundColor
   // 让 Monaco 通过 CSS 类应用样式
   ```

## 三、问题清单

### 3.1 样式不一致问题

| 问题 | 位置 | 现状 | 建议 |
|------|------|------|------|
| 透明度不统一 | 多处 | 35%、59%、15% 混用 | 统一定义为 `--accent-opacity-35: color-mix(in srgb, var(--accent-color) 35%, transparent)` |
| 内联样式 vs CSS 变量 | PropertyPanel, MonacoEditor | 混用 | 全部迁移到 CSS 变量 |
| 颜色格式不一致 | 部分使用 hex，部分使用 rgb | 混用 | 统一使用 CSS 变量 |

### 3.2 缺失的 CSS 变量定义

建议在 `global.css` 中添加以下派生变量：

```css
:root {
  /* 主色派生变量 - 统一透明度 */
  --accent-color: #FCA506;
  --accent-light: color-mix(in srgb, var(--accent-color) 15%, white);
  --accent-opacity-10: color-mix(in srgb, var(--accent-color) 10%, transparent);
  --accent-opacity-15: color-mix(in srgb, var(--accent-color) 15%, transparent);
  --accent-opacity-35: color-mix(in srgb, var(--accent-color) 35%, transparent);
  --accent-opacity-50: color-mix(in srgb, var(--accent-color) 50%, transparent);
  
  /* 主色文字颜色（用于主色背景上的文字） */
  --accent-text: white; /* 或根据主色亮度动态计算 */
}
```

## 六、技术建议

### 6.1 CSS 变量命名规范

```css
/* 推荐命名 */
--accent-color              /* 主色原色 */
--accent-light              /* 主色浅色（+15% 白） */
--accent-opacity-{10|15|35|50}  /* 主色透明度 */
--accent-text               /* 主色背景上的文字颜色 */
```

### 6.2 透明度统一标准

- **10%**：悬停效果（极轻微）
- **15%**：悬停效果（明显）、浅色背景
- **35%**：激活状态背景、选中状态背景
- **50%**：边框、强调元素

### 6.3 性能优化建议

1. **避免内联样式**：内联样式会阻止浏览器 CSS 优化
2. **使用 CSS 类切换**：而非动态修改 style
3. **利用 CSS 变量继承**：减少重复定义
