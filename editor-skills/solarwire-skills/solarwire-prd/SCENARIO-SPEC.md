# SolarWire 场景规格

## Mobile H5 场景

### 基本规格
| 属性 | 值 |
|------|-----|
| 默认宽度 | 375px |
| 默认高度 | 812px |
| 安全区域 | 顶部 44px，底部 34px |
| 导航栏高度 | 44px |
| 标签栏高度 | 49px |

### 常见布局模式
- 列表页：标题栏 + 搜索 + 列表 + 底部导航
- 详情页：标题栏 + 内容区 + 底部操作
- 表单页：标题栏 + 表单区 + 提交按钮

### 交互规范
- 点击区域 ≥ 44x44px
- 手势操作需有视觉反馈
- 表单提交需防重复提交

---

## Web 场景

### 基本规格
| 属性 | 值 |
|------|-----|
| 默认宽度 | 1440px |
| 默认高度 | 900px |
| 内容最大宽度 | 1200px |
| 侧边栏宽度 | 240px（如有） |

### 常见布局模式
- 首页：顶部导航 + Hero 区 + 内容网格 + 页脚
- 列表页：顶部导航 + 筛选区 + 列表/表格 + 分页
- 详情页：顶部导航 + 面包屑 + 详情区 + 相关推荐

### 交互规范
- Hover 状态反馈
- 表格支持排序和筛选
- 响应式断点：768px、1024px、1440px

---

## Admin 场景

### 基本规格
| 属性 | 值 |
|------|-----|
| 默认宽度 | 1440px |
| 默认高度 | 900px |
| 侧边栏宽度 | 240px |
| 顶部栏高度 | 64px |

### 常见布局模式
- 控制台：侧边栏 + 顶部栏 + 统计卡片 + 图表 + 最近活动
- 列表管理：侧边栏 + 顶部栏 + 筛选区 + 数据表格 + 分页
- 表单编辑：侧边栏 + 顶部栏 + 表单区 + 保存/取消按钮

### 交互规范
- 数据表格支持批量操作
- 表单验证实时反馈
- 操作需有确认弹窗（删除等危险操作）
- 支持快捷键操作

---

## 场景对比

| 维度 | Mobile | Web | Admin |
|------|--------|-----|-------|
| 主要交互 | 触控 | 鼠标 + 键盘 | 鼠标 + 键盘 |
| 导航方式 | 底部标签栏 | 顶部导航 | 侧边栏 |
| 数据密度 | 低 | 中 | 高 |
| 页面数量 | 通常较多 | 中等 | 通常较少 |
| 表单复杂度 | 简单 | 中等 | 复杂 |
